vi

i
enter insert mode
Esc
return to normal mode
:w
write (save) the file
:q
quit
:wq
save and quit
dd
delete (cut) the current line
yy
yank (copy) the current line
p
paste after the cursor
u
undo last change
/word
search for "word"
n
repeat search in same direction
N
repeat search in opposite direction
G
go to last line
gg
go to first line
J
join current line with next
. (dot)
repeat last command
:set nu
show line numbers
:set nonu
hide line numbers
:%s/foo/bar/g
replace "foo" with "bar" globally
:vsplit file
open file in vertical split
:split file
open file in horizontal split
ctrl-w h/j/k/l
move between splits
x
delete character under cursor
X
delete character before cursor
dw
delete word from cursor forward
daw
delete entire word (including space)
de
delete to end of word
d$
delete to end of line
d0
delete to start of line
D
delete from cursor to end of line
dG
delete from current line to end of file
dgg
delete from current line to start of file
d/word
delete until "word" is found
